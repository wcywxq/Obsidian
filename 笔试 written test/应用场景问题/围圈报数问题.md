有 30 个小孩儿，编号从 1-30，围成一圈依此报数，1、2、3 数到 3 的小孩儿退出这个圈， 然后下一个小孩 重新报数 1、2、3，问最后剩下的那个小孩儿的编号是多少?

```javascript
function childNum(n, m){
    let res = 0
    for (let i = 2; i <= n; i++) {
        res = (res + m) % i
    }
    return res
}
childNum(30, 3)
```
