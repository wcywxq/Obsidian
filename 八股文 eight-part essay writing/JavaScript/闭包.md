闭包指的是有权访问另一个函数作用域中变量的函数，创建闭包最常见的方式就是：在一个函数内部创建另外一个函数(本质就是函数作用域链中保存着对外部函数变量对象的引用) <a name="XfPJi"></a>

###

<a name="Aq4uw"></a>

## 案例

- 修改以下案例

```javascript
var data = []

for (var i = 0; i < 3; i++) {
    data[i] = function () {
        console.log(i)
    }
}

data[0]() // 3
data[1]() // 3
data[2]() // 3
```

- 改良为闭包后

```javascript
var data = []

for (var i = 0; i < 3; i++) {
    data[i] = (function (i) {
        return function () {
            console.log(i)
        }
    })(i)
}

data[0]() // 0
data[1]() // 1
data[2]() // 2
```
