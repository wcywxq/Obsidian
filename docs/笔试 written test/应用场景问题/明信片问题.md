聊一个明信片问题吧，你有一个明信片，要寄给一个陌生人，有几个问题

- 你怎么保证不被快递员知道内容？（包上信封,这里埋了个坑，没有联想到另一篇看过的鸽子通信里面，用箱子锁上更加可靠）
- 快递员把信封拆掉呢？（在信的内容里添加线索，在信封外加上线索进行一个解密对应，当时没get到面试官针对上面的提示）
- 信封被换掉了呢？（找权威的邮局加上印章）
- 陌生人怎么知道哪些是权威的印章呢？（人手一份邮局的对照表）
  - 那印章发生了更新呢？（邮局通知或者自己去更新）
