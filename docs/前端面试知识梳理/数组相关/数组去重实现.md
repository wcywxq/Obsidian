## 利用 filter + indexOf 去重

```javascript
function uniq(arr) {
	return arr.filter((item, index, array) => array.indexOf(item) === index)
}
```

<a name="bi8jJ"></a>

## 利用 reduce  + includes 去重复

```javascript
function uniq(arr) {
	return arr.reduce((prev, cur) => (prev.includes(cur) ? prev : [...prev, cur], []))
}
```

<a name="Yutfx"></a>

## 利用 Set 数据结构去重

```javascript
function uniq(arr) {
	return Array.from(new Set(arr))
}
```
