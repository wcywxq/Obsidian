## 常用正则表达式

```javascript
// （1）匹配 16 进制颜色值
var regex = /#([0-9a-fA-F]{6}|[0-9a-fA-F]{3})/g;

// （2）匹配日期，如 yyyy-mm-dd 格式
var regex = /^[0-9]{4}-(0[1-9]|1[0-2])-(0[1-9]|[12][0-9]|3[01])$/;

// （3）匹配 qq 号
var regex = /^[1-9][0-9]{4,10}$/g;

// （4）手机号码正则
var regex = /^1[34578]\d{9}$/g;

// （5）用户名正则
var regex = /^[a-zA-Z\$][a-zA-Z0-9_\$]{4,16}$/;
```

### url 参数转对象

```javascript
function getParams(url) {  
  const reg = /(?:&|\/?)(\w+)=([^&$]+)/g
  const params = {}
  while(result = reg.exec(url)) {
  	params[result[1]] = result[2]
  }
  return params
}
```

### 数字价格千分位分割

描述：将 123456789 变成 123,456,789

```javascript
'123456789'.replace(/(?!^)(?=(\d{3})+$)/g, ',')
```

描述：将手机号 183xxxxxxxx 转化成 183-xxxx-xxxx

```javascript
'18312345678'.replace(/(?=(\d{4})+$)/g, '-')
```

### 字符串驼峰化

描述：

- foo Bar => fooBar
- foo-bar---- => fooBar
- foo\_bar\_\_\_\_ => fooBar

```javascript
const camelCase = str => {
	const camelCaseRegex = /[-_\s]+(.)?/g
  return str.replace(camelCaseRegex, (match, char) => {
  	return char ? char.toUpperCase() : ''
  })
}
```

### 将字符串首字母转化为大写，剩下为小写

描述：hello world => Hello World

```javascript
const capitalize = str => {
	const capitalizeRegex = /(?:^|\s+)\w/g
  return str.toLowerCase().replace(capitalizeRegex, match => {
  	return match.toUpperCase()
  })
}
```

### 提取连续重复的字符

描述：将有重复的字符串提取出来，例如：12323454545666，提取\['23', '45', '6']

```javascript
const collectRepeatStr = str => {
	let repeatStrs = []
  const repeatRegex = /(.+)\1+/g
  str.replace(repeatRegex, ($0, $1) => {
  	$1 && repeatStrs.push($1)
  })
  return repeatStrs
}
```

### 实现一个 trim 函数

描述：去除字符串的首尾空格

```javascript
// 去除空格法
const trim = str => str.replace(/^\s*|\s*$/g, '')
// 提取非空格法
const trim = str => str.replace(/^\s*(.*?)\s*$/g, '$1')
```
