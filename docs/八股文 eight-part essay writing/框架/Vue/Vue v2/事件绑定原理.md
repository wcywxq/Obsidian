## 组件事件

通过 vue 自定义的 $on 实现；在组件上使用原生事件，需要加 .native 修饰符(相当于父组件中把子组件当作普通的 html 标签, 然后加上原生事件)

on、emit 是基于发布订阅模式的, 维护一个事件中心, on 的时候将事件按名称存在事件中心里，即订阅者。然后 emit 将对应事件发布，去执行事件中心里对应的监听器。
