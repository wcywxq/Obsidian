- `~`波浪号，匹配最新补丁版本号，即版本号的第三个数字，例如`~3.0.0`就会匹配 `3.0.x` 版本，将在 `3.1.0` 停止
- `^`插入符号，匹配次要的版本号，即版本号的第二个数字，例如`^3.0.0`就会匹配任何 `3.x.x` 版本，将在 `4.0.0` 停止
- `>、<、>=、<=`比较运算符，匹配的就是这个区间的版本，例如`>3.0.0 <= 3.1.4`，就会匹配这个区间的版本号
