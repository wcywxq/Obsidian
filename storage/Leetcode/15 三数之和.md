[15. 三数之和](https://leetcode.cn/problems/3sum) <a name="B3liy"></a>

## 题目描述

```javascript
//给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != 
//k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。请 
//
// 你返回所有和为 0 且不重复的三元组。 
//
// 注意：答案中不可以包含重复的三元组。 
//
// 
//
// 
//
// 示例 1： 
//
// 
//输入：nums = [-1,0,1,2,-1,-4]
//输出：[[-1,-1,2],[-1,0,1]]
//解释：
//nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。
//nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。
//nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 。
//不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。
//注意，输出的顺序和三元组的顺序并不重要。
// 
//
// 示例 2： 
//
// 
//输入：nums = [0,1,1]
//输出：[]
//解释：唯一可能的三元组和不为 0 。
// 
//
// 示例 3： 
//
// 
//输入：nums = [0,0,0]
//输出：[[0,0,0]]
//解释：唯一可能的三元组和为 0 。
// 
//
// 
//
// 提示： 
//
// 
// 3 <= nums.length <= 3000 
// -10⁵ <= nums[i] <= 10⁵ 
// 
//
// Related Topics 数组 双指针 排序 👍 6522 👎 0
```

## 题解

<a name="tlnTv"></a>

### 数组 + 双指针

```javascript
/**
 * 数组 + 双指针
 * @param {number[]} nums
 * @return {number[][]}
 */
var threeSum = function (nums) {
    let res = []
    let len = nums.length
    if (len < 3) return res

    nums.sort((a, b) => a - b)

    for (let i = 0; i < len - 2; i++) {
      // 第一个数 > 0 不成立
      if (nums[i] > 0) {
        break
      }
      // 去重
      if (i > 0 && nums[i] === nums[i - 1]) {
        continue
      }

      let l = i + 1
      let r = len - 1
      while (l < r) {
        let sum = nums[i] + nums[l] + nums[r]

        if (sum === 0) {
          res.push([nums[i], nums[l], nums[r]])
          // 去重
          while (nums[l] === nums[l + 1]) {
            l++
          }
          while (nums[r] === nums[r - 1]) {
            r--
          }
          l++
          r--
        } else if (sum < 0) {
          l++
        } else {
          r--
        }
      }
    }
    return res
}
```
